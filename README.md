# In Clean India – AI Smart Garbage & Water Pollution Complaint Management System

> **Report • Detect • Track • Clean**

An AI-powered Smart City platform that helps citizens report garbage, dirty water, blocked drains, overflowing dustbins, potholes, sewage overflow, and other public cleanliness issues. The platform enables municipal authorities to manage complaints efficiently and allows citizens to track their complaints in real time.

## 📖 Project Overview

Clean India is an AI-powered complaint management system designed to improve public cleanliness and environmental safety.

Citizens can upload images of public cleanliness issues, and AI analyzes the image to identify the issue type, estimate its priority, and generate a complaint automatically.

The complaint is sent to the Municipal Administration Dashboard, where officials can review, assign, update, and resolve complaints.

## 🚀 How the Application Works

### Step 1
Citizen registers or logs into Clean India.

### Step 2
User clicks **Report Complaint**.

### Step 3
Upload an image of the issue.

### Step 4
AI analyzes the uploaded image and identifies the issue.

### Step 5
User enters location, address, and description.

### Step 6
The system generates a unique Complaint ID automatically.

Example:

**CC202600001**

### Step 7
Complaint is stored in the database.

### Step 8
Municipal Admin receives the complaint.

### Step 9
Admin updates complaint status:

- Pending
- In Progress
- Resolved

### Step 10
User receives status updates.

### Step 11
User tracks the complaint using:

- Complaint ID
- Mobile Number
- Email Address
